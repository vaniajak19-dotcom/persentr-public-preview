# Persentr Expert Hub Public Preview Package

Эта папка — безопасный статический пакет для публичной публикации через GitHub Pages.

Её можно скопировать целиком в отдельный публичный GitHub-репозиторий и включить GitHub Pages из корня репозитория или из выбранной папки публикации.

## Состав пакета

- `index.html` — стартовая страница публичного preview;
- `roadmap_dashboard.html` — статический dashboard дорожной карты;
- `roadmap_status.json` — публичные данные для dashboard;
- `.nojekyll` — отключение Jekyll-обработки GitHub Pages.

## Что намеренно не входит

Пакет не содержит внутренние материалы проекта, включая:

- `02_expertise_units/`;
- `12_mvp_control_desk/`;
- `ROADMAP.md`;
- внутренние отчёты;
- source registry;
- рабочие документы Persentr.

## Как использовать в отдельном публичном репозитории

1. Создайте отдельный публичный GitHub-репозиторий.
2. Скопируйте содержимое этой папки в корень нового репозитория.
3. Закоммитьте файлы и отправьте их на GitHub.
4. Откройте **Settings → Pages**.
5. Включите публикацию из нужной ветки и папки, например `main` / `/root`.
6. Откройте опубликованный URL GitHub Pages.

Dashboard загружает `roadmap_status.json` из той же папки, поэтому пакет можно переносить без дополнительных зависимостей.
